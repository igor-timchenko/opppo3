МебельДомТорг ->
Создаём новый отчёт, называем его "ОтчетПоКонтрагентам"
Нажимаем на кнопку "Открыть схему компановки данных" -> Готово -> Добавить набор данных запрос -> Конструктор запроса -> 
Переносим из базы данных Справочники -> Контрагенты в таблицы -> Развернём контрагенты -> Выбыраем Код и Наименование в поля -> Нажимаем на кнопку "Ок"
ВЫБРАТЬ
	Контрагенты.Код КАК Код,
	Контрагенты.Наименование КАК Наименование
ИЗ
	Справочник.Контрагенты КАК Контрагенты

Переходим на последнюю закладку "настройки" -> Открыть конструктор настроек -> Выбираем "Список" ->
Код и Наименование перенести в поле -> Нажимаем на кнопку "Ок" -> Создаться вариант отчета
Переходим в подсистемы и выбираем "НСИ"

Запускаем программу -> Переходим во вкладку "НСИ" -> Выбираем отчеты -> Отчет по Кантрагентам -> Сформировать -> Показывается таблица

МебельДомТорг ->
Создаём новый отчёт, называем его "ПоставщикиНоменклатуры"
Включаем его сразу в подсистему "НСИ"
Нажимаем на кнопку "Открыть схему компановки данных" -> Готово -> Добавить набор данных запрос -> Конструктор запроса -> 
Переносим из базы данных Справочники -> Номенклатура в таблицы -> Развернём Номенклатуру -> Выбыраем Ссылка и Основной поставщик в поля -> Нажимаем на кнопку "Ок"

Конструктор запроса -> Объединение/Псевдонимы -> Ссылка меняем на Номенклатура -> "OK"

ВЫБРАТЬ
	Номенклатура.Ссылка КАК Номенклатура,
	Номенклатура.ОсновнойПоставщик КАК ОсновнойПоставщик
ИЗ
	Справочник.Номенклатура КАК Номенклатура

Переходим на последнюю закладку "настройки" -> Открыть конструктор настроек -> Номенклатура и ОсновнойПоставщик перенести в вкладку поле -> Нажать "Ок"

Запускаем программу -> Переходим во вкладку "НСИ" -> Выбираем отчеты -> Поставщики Номенклатуры -> Сформировать -> Показывается таблица

Запускаем программу -> Переходим во вкладку "НСИ" -> Выбираем отчеты -> Поставщики Номенклатуры -> Выбрать вариант -> По группировкам -> Сформировать

Комментарии в програмном модуле
  Предназначение
    Дать текстовое описание алгоритму
    При изменении кода вынести старый вариант в комментарий

  Программный код в комментариях не будет выполняться

  Установка комментария

либо удали слэшы
либо выделить программный код и нажать на кнопку удалить комментарий


Переменные и оператор присваивания
  В переменых хранятся значения любых типов данных

  Требования к наименованию переменной
    Состоит из букв, цифр и знаков подчеркивания

  Оператор присваивания
    Переменная = Значение

  Объявление переменных, обращения к переменным

МебельДомТорг -> Открыть модуль управляемого приложения ->

//Сообщить("Привет, мир!!!);
//Сообщить("Hello, world!!!);

СчастливоеЧисло = 777;
Сообщить(СчастливоеЧисло);
Выведится 777

СчастливоеЧисло = 777;
Сообщить(СчастливоеЧисло);
СчастливоеЧисло = 888;
Сообщить(СчастливоеЧисло);
Выведится 777 потом 888

Напишем 
Перем СчастливоеЧисло;
Вывод не изменется


Условный оператор
  Операции сравнения
    =,>,>=,<,<=,<>

  Конструкция Если <Условие> Тогда... КонецЕсли

  Ветка невыполнения условия - Иначе

  Множественные условия

СчастливоеЧисло = 777;
Сообщить(СчастливоеЧисло);
СчастливоеЧисло = 888;
Сообщить(СчастливоеЧисло);

Если СчастливоеЧисло = 777 Тогда
  Сообщить("Всё правильно!!!);
ИначеЕсли СчастливоеЧисло = 888 Тогда
  Сообщить("Восьмёрки");
Иначе 
  Сообщить("Ошибка!!!")
КонецЕсли;


Циклические операторы
  Цикл по счётчику
    Для <Переменная> = <Начальное значение> По <Конечное значение> Цикл... КонецЦикла
  Пример цикла

  Цикл по выполнению условия
    Пока <Условие> Цикл... КонецЦикла

  Прерывания цикла

  Переход к следующей итерации цикла

Для Счетчик = 6 По 20 Цикл
  Сообщить(Счетчик);
  Сообщить(Счетчик * Счетчик);
КонецЦикла

Цикл вида пока

ИскомоеЧисло = 1;
Пока ИскомоеЧисло*ИскомоеЧисло < 30000 Цикл
  ИскомоеЧисло = ИскомоеЧисло + 1;
КонецЦикла;
Сообщить(ИскомоеЧисло - 1);

В цикле есть оператор:
  Прервать;
и
  Продолжить;
